# Onionskey

This is a fork of [Misskey](https://github.com/misskey-dev/misskey) that runs over Tor.

## Features

Based on Misskey v13.14.2.

- Tor support
  - SOCKS5 proxy support
  - Federation with other ActivityPub instances on Tor
- Some bug fixes
